# Web SmartCard Typing (Draft)

This is [Web SmartCard](https://wicg.github.io/web-smart-card/) typing.

## Usage

Installation:

```bash
npm install @nicelabs/web-smartcard-types
```

`tsconfig.json` Configuration:

```jsonc
{
  "compilerOptions": {
    // ...
    "types": ["@nicelabs/web-smartcard-types"]
    // ...
  }
}
```
